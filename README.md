# Vela Textile DPP Prototype

Static one-page Digital Product Passport prototype for the TraceWeave practical case.

## Files

- `index.html` — full one-page site with embedded CSS and JavaScript
- `vercel.json` — Vercel routing fallback
- `README.md` — project notes

## Deployment

Upload these three files directly to the root of your GitHub repository:

```text
your-repository/
  index.html
  vercel.json
  README.md
```

Then deploy with Vercel using:

```text
Framework Preset: Other
Build Command: leave empty
Output Directory: leave empty or .
Root Directory: leave empty
```

## Scope

The page represents only the 9 DPP blocks requested in the case:

1. Identification
2. Origin & production
3. Composition
4. Traceability
5. Durability & repairability
6. Environmental impact
7. End of life
8. Compliance
9. Metadata

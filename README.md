# Vela Textile DPP Prototype

Static HTML prototype for the TraceWeave practical case.

## Deploy on Vercel

1. Put `index.html` at the root of the GitHub repository.
2. Import the GitHub repository into Vercel.
3. Framework preset: Other / Static.
4. Build command: leave empty.
5. Output directory: leave empty or use `.`.
6. Deploy.

The homepage must be called exactly `index.html` in lowercase.
